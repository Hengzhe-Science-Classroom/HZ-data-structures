# Data Structures · Interactive

**Arrays, trees, graphs, hash tables — organizing data efficiently**

**Status:** Planned

Part of [Hengzhe's Science Classroom](https://hengzhe-science-classroom.github.io/hub/) · Tier 3 · Undergraduate

## What You'll Learn

- Arrays, linked lists, stacks, queues
- Trees: BST, AVL, red-black, B-trees
- Heaps and priority queues
- Hash tables and collision resolution
- Graphs: representation, traversal
- Sorting and searching algorithms

## References

- Robert Sedgewick & Kevin Wayne, *Algorithms* (4th ed., Addison-Wesley)
- Mark Allen Weiss, *Data Structures and Algorithm Analysis in Java* (3rd ed.)
- Steven S. Skiena, *The Algorithm Design Manual* (3rd ed., Springer)

## Features (when built)

- **Interactive visualizations** — every concept comes alive with draggable, animated demos
- **KaTeX math rendering** — rigorous notation throughout
- **Exercises with hints & solutions** — from computational to proof-based
- **Progress tracking** — saved locally in your browser
- **Zero dependencies** — just open the link and learn
